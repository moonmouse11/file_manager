# file_manager
Simple file manager in Java for studying. Bash clone.
Work in Linux. I don't test it in Windows yet.
Console output in Russian.

## Commands

* `ls`
* `ll` 
* `cp` 
* `touch` 
* `cat` 
* `cd` 
* `mkdir` 
* `rmdir` 
* `rm`
* `pwd` 
* `echo` 
* `history`
* `help` 
* `exit`
